## Shemday: The Glow Within ✨

Selamat datang di **Shemday: The Glow Within**, sebuah mini-game interaktif berbasis web!  
Game ini aku rancang untuk menghadirkan pengalaman hangat dan inspiratif melalui pertanyaan sederhana, amplop animasi khusus buat kamu.

--

## 🕹️ Cara Main

1. Buka `index.html` di browser (Chrome, Firefox, Edge, dll).  
2. Baca **penjelasan awal Shemday** (ditampilkan selama 10 detik).  
3. Jawab pertanyaan pertama: “Kalo kamu terlahir kembali, kamu pilih jadi apa?”  
   - Pilihan: Kucing 🐱 atau Anjing 🐶  
4. Pilih sub-pertanyaan (misal: warna kucing atau jenis anjing).  
5. Klik amplop yang muncul untuk membuka **rahasia**. 

---

## 🛠️ Teknologi

- **HTML5** – Struktur halaman dan konten  
- **CSS3** – Style, animasi amplop, background, tombol glow  
- **JavaScript** – Logika interaktif, pertanyaan & sub-pertanyaan, pemilihan quotes, efek fade-in
